# JFA 緑プロジェクト - プロトタイプ

**「サッカーと生きる、すべての人のために」**

日本サッカー協会（JFA）の次世代プラットフォームのプロトタイプです。

## 🎯 このプロトタイプで体験できること

### Magic Moment: 成長の可視化

- 📈 **成長グラフ** - 4月から10月までの7ヶ月間の成長を視覚化
  - ドリブル、パス、シュート、ディフェンス、フィジカルの5つのスキル
  - アニメーション付きのインタラクティブなグラフ

- 🎯 **スキルレーダーチャート** - 現在のスキルレベルを一目で把握
  - 各スキルの成長量も表示（+10など）

- ⚽ **試合履歴** - 直近5試合の詳細な記録
  - 試合結果、スコア、個人成績
  - コーチからのフィードバック
  - 5段階評価

- 🏆 **達成バッジ** - モチベーションを高める達成システム
  - 初ゴール、記録マスター、今シーズン5ゴール、成長中

## 🚀 起動方法

プロトタイプは既に起動しています：

**アクセスURL:** http://localhost:3001

ブラウザで上記URLを開いてください。

### 初めて起動する場合

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

## 🛠 技術スタック（2025年のベストプラクティス）

- **Next.js 15** - App Router、Turbopack（開発時の高速ビルド）
- **React 19** - 最新のReactフレームワーク
- **TypeScript** - 型安全性
- **Tailwind CSS v3** - ユーティリティファーストCSS
- **Recharts** - アニメーション付きグラフライブラリ
- **date-fns** - 日付フォーマット（軽量）
