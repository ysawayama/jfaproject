## 🎯 プロジェクト概要

このポータルサイトは、JFA（日本サッカー協会）SAMURAI BLUE A代表チームの運営を効率化し、内部管理と外部公開を一元化するために設計されたハイブリッド型プラットフォームです。

### 🌟 コンセプト

- **内部管理**：監督・コーチ陣、選手が日々のトレーニング、試合分析、コミュニケーションを行う
- **外部公開**：メディア関係者、サポーターが試合情報やニュース、選手情報にアクセス
- **役割ベース権限管理**：ユーザーの役割に応じた情報表示と機能制限

### ✨ 特徴

- 🎨 **60-30-10カラールール** - サムライブルーをベースとした洗練されたデザインシステム
- 📱 **完全レスポンシブ** - モバイル、タブレット、デスクトップ全対応
- 🔐 **ロールベースアクセス制御** - 5つの役割（Admin, Coach, Player, Media, Fan）
- 🌐 **多言語対応準備** - 日本語・英語の切り替え基盤
- ⚡ **高速パフォーマンス** - Next.js 15 + Turbopack採用

## 🚀 主要機能

### 📅 1. スケジュール管理

- **月/週/リスト表示切り替え** - 柔軟なビュー切り替え
- **イベント詳細モーダル** - 試合、トレーニング、ミーティング、メディア対応の詳細表示
- **RSVP機能** - ワンクリック出欠確認（参加/不参加/未定）
- **カレンダーエクスポート** - iCal形式でのエクスポート対応
- **Googleマップ連携** - 会場の地図表示

### 📊 2. 試合分析

- **試合詳細カード** - スコア、統計データ、ラインナップ、タイムライン
- **タブ切り替え** - 概要/統計データ/ラインナップ/タイムライン
- **統計データ可視化** - ポゼッション、シュート数、パス成功率などを視覚的に表示
- **ラインナップ表示** - フォーメーション図付き
- **試合履歴** - 直近の試合と今後の試合一覧

### 👥 3. チームメンバー管理

- **選手グリッド表示** - 背番号、ポジション、基本情報
- **ポジションフィルター** - GK/DF/MF/FWで絞り込み
- **ソート機能** - 背番号順/名前順
- **選手詳細ページ** - プロフィール、統計データ、直近のフォーム

### 📹 4. メディアライブラリ

- **メディアグリッド表示** - 動画、写真、資料を一元管理
- **タイプフィルター** - 動画/写真/資料で絞り込み
- **タグ検索** - タグベースの検索機能
- **アップロード機能** - ドラッグ&ドロップ対応（UI実装済み）
- **閲覧数トラッキング** - 人気コンテンツの把握

### 💬 5. チームコミュニケーション

- **チャンネルベース** - Slack風のチャンネル管理
- **権限制御** - 全体通知、コーチ専用、選手グループなど
- **リアルタイムメッセージ** - 迅速なコミュニケーション
- **ファイル添付・絵文字対応** - リッチなメッセージ機能

## 💻 ローカル開発環境セットアップ

### 前提条件

- Node.js 18.17以上
- npm または yarn

### インストール手順

```bash
# リポジトリのクローン
git clone https://github.com/your-org/jfa-team-portal.git
cd jfa-team-portal

# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

### アクセス

ブラウザで以下のURLを開く：
```
http://localhost:3000
```

### ビルド＆本番環境

```bash
# プロダクションビルド
npm run build

# プロダクションサーバー起動
npm start
```

## 🎮 デモモード

このポータルはデモモードで動作します。ユーザーはロール選択モーダルで役割を選び、その役割に応じた機能とデータにアクセスできます。

### ロール選択の流れ

1. 初回アクセス時に「ロール選択モーダル」が表示
2. 5つの役割から1つを選択
   - 👨‍🏫 監督・コーチ
   - ⚽ 選手
   - 👔 JFA職員
   - 📸 メディア
   - 🎉 サポーター
3. 名前を入力して「ポータルに入る」
4. 選択した役割に応じたダッシュボードが表示される

### デモデータ

- **スケジュール**: 2025年11月のイベント16件
- **試合データ**: 直近3試合、今後2試合
- **選手データ**: 12名の代表選手
- **メディア**: 動画、写真、資料8件
- **メッセージ**: チャンネル4つ、メッセージ3件
